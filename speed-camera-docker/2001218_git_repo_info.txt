[ 구현 내용 ]

- Docker Compose를 이용한 n8n + MySQL + Grafana 연동
- 주제: 80km/h 제한 차량 단속 카메라 시뮬레이션
- 속도 범위: 60 ~ 100 km/h (80 ± 20)
- 5초마다 차량 2대씩 속도/차종/과속여부 자동 기록
- Grafana 대시보드: 속도 추이 그래프, 과속 비율, 단속 기록 테이블
